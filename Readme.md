# 社用車予約管理システム(Google Apps Script)

Googleスプレッドシート + Google Apps Script(GAS)だけで動く、社用車の予約管理システムです。
無料で、自分のGoogleアカウントの中だけで完結して動かせます。

## できること

- **マスタ画面**:社用車の登録(車種名・ナンバー・定員・利用可能時間帯)、車検・点検などの使用不可期間の登録
- **予約確認画面**:月間カレンダーで予約件数を確認し、日付を選ぶと全車両の空き状況を一覧表示。空いている枠をクリックするとそのまま予約登録画面へ
- **予約登録画面**:利用開始日時・返却日時(日をまたぐ利用にも対応)、利用者名、目的、行き先を入力して予約登録。予約者名をクリックすれば編集・削除も可能

## 必要なファイル

このリポジトリには以下のファイルが含まれています。

| ファイル名 | 役割 |
|---|---|
| `Code.gs` | サーバーサイドのロジック(データの読み書き、画面の振り分けなど) |
| `マスタ画面.html` | 車両登録・使用不可期間管理の画面 |
| `予約確認画面.html` | カレンダー・空き状況確認の画面 |
| `予約登録画面.html` | 予約入力の画面 |

## セットアップ手順

### 1. スプレッドシートを新規作成する

Googleドライブで「新規」→「Google スプレッドシート」を選び、空のスプレッドシートを作成します。
(シート名やスプレッドシート名は何でも構いません)

### 2. Apps Scriptエディタを開く

スプレッドシート上部メニューの「拡張機能」→「Apps Script」を選択します。

### 3. コードを貼り付ける

1. デフォルトで作成されている `コード.gs`(または `Code.gs`)の中身を全て削除し、このリポジトリの `Code.gs` の内容を貼り付けます。
2. 左側のファイル一覧の「+」ボタン→「HTML」を選び、ファイル名を **`マスタ画面`** として作成します(拡張子の `.html` は入力不要、エディタが自動で付けます)。中身をこのリポジトリの `マスタ画面.html` の内容で置き換えます。
3. 同様に **`予約確認画面`**、**`予約登録画面`** という名前でHTMLファイルを追加し、それぞれ対応する内容を貼り付けます。
4. ファイル名は上記と完全に一致させてください(前後のスペースや拡張子の重複に注意)。
5. 保存します(Ctrl+S またはフロッピーアイコン)。

### 4. シートを初期化する

1. Apps Scriptエディタ上部の関数選択プルダウンで **`setupSheets`** を選択します。
2. 「実行」ボタンをクリックします。
3. 初回は「承認が必要です」という画面が出ます。「権限を確認」→ Googleアカウントを選択 →「詳細」→「(プロジェクト名)に移動」→「許可」の順に進めてください(自分が作ったスクリプトなので安全です)。
4. 実行が完了すると、スプレッドシートに「車両マスタ」「使用不可期間」「予約データ」の3つのシートが自動的に作成されます(元からあった不要なシートは自動的に削除されます)。

### 5. Webアプリとしてデプロイする

1. Apps Scriptエディタ右上の「デプロイ」→「新しいデプロイ」をクリック
2. 歯車アイコン(種類の選択)→「ウェブアプリ」を選択
3. 「次のユーザーとして実行」→ **自分** を選択
4. 「アクセスできるユーザー」→ 用途に応じて選択(社内で共有するなら「組織内の全員」、自分だけで使うなら「自分のみ」など)
5. 「デプロイ」をクリック
6. 表示されたウェブアプリのURLにアクセスすると、予約確認画面が開きます

### 6. 使い始める

1. 表示されたURLを開く(初期表示は予約確認画面です)
2. 「マスタ画面」から社用車を登録
3. 予約確認画面で空き状況を確認しながら予約を登録

## コードを修正した場合

Apps Scriptエディタでコードを編集して保存しただけでは、公開済みのWebアプリURLには反映されません。
「デプロイ」→「デプロイを管理」→ 編集(鉛筆アイコン)→ バージョンを「新バージョン」に変更 →「デプロイ」の手順で再デプロイしてください(URLは変わりません)。

## 注意事項

- 予約は30分単位です
- 利用者の認証機能はありません(フリー入力の利用者名で管理する簡易的な仕組みです)。厳密な本人確認が必要な用途には向きません
- 車両ごとに「利用可能時間帯」を制限できますが、制限を設定した車両は日をまたぐ予約はできません。日をまたぐ利用(出張など)をしたい場合は、マスタ画面で該当車両を「時間帯を制限しない」設定にしてください
